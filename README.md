# doneliezer.com

Personal portfolio for DonEli Baize — senior iOS engineer, U.S. patent
holder, former founding engineer of PersonaFi.

Hand-written HTML/CSS, no build step, no framework. Hosted on GitHub
Pages at the apex domain. Each page is self-contained: design tokens
in a `:root` block, dark mode via `prefers-color-scheme`, and the same
typography (Fraunces / Manrope / JetBrains Mono) and palette across
the five pages.
